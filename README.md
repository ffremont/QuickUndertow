Syntaxe : java [-options] -jar xxxxx.jar

où options :
    -D<name>=<value>

            server.host     : nom du serveur
            server.port     : port du serveur
            server.listing  : liste les contenus des répertoires
            server.base     : répertoire racine du serveur
